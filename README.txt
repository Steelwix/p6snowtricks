1 - Create a folder in which to call the project

2 - Import the project from Github with the "pull" command

3 - Recover the snowtricks.SQL file

4 - Import the SQL file into your database system (wampserver for example, or any other system you use)

5 - In the .env file, modify the DATABSE_URL field to make the link between the project and your database 
(the default format is: "mysql://db_user:db_password@127.0.0.1:3306/db_name?serverVersion= 5.7&charset=utf8mb4"
